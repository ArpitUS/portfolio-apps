# AutoFlow Engine

**Description:**  
Drag-and-drop workflow automation for SMEs using Go and Temporal.io.

**Tech Stack:**  
- Frontend: React  
- Backend: Go (Gin), Temporal.io  
- DB: PostgreSQL

**Core Features:**  
- Visual workflows  
- Event triggers  
- Logs & analytics

**Setup Instructions:**  
1. go run main.go  
2. npm run dev
